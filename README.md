
<h1>Calculadora Com JavaScript, HTML e CSS</h1>

<p>
Este projeto é uma calculadora simples desenvolvida em JavaScript, que permite realizar operações matemáticas básicas de forma interativa. A interface da calculadora é construída utilizando HTML e CSS, enquanto a lógica de cálculo é implementada em JavaScript.

Funcionalidades Principais:
* Interação via Botões: O usuário pode clicar em botões numéricos e operacionais para construir expressões matemáticas. A calculadora responde aos cliques e exibe os valores no display.

* Tecla Enter: Além dos botões, o usuário pode pressionar a tecla "Enter" para realizar o cálculo, proporcionando uma experiência de uso mais fluida.

* Limpeza e Deleção: O usuário pode limpar todo o display ou remover o último caractere inserido, permitindo correções rápidas.

* Validação de Entrada: O sistema valida a expressão inserida para garantir que apenas números e operadores válidos sejam utilizados. Caso a entrada seja inválida, uma mensagem de alerta é exibida.

* Impedimento de Zeros Iniciais: A calculadora evita que um zero seja inserido no início de uma expressão, garantindo uma entrada numérica correta.

</p>
